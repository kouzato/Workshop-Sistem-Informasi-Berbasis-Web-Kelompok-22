1. tag meta Compat digunakan untuk menangani permasalahan terkait web browser IE / Internet Explorer. tag ini memaksa IE untuk tidak masuk ke mode competible view. competible view merupakan mode yang aktif ketika halaman web diakses dari komputer lokal atau intranet

2. tag form digunakan untuk mendefinisikan pembuatan form

3. tag input typpe="button" digunakan untuk membuat tombol

4. tag cap digunakan untuk membuat caption atau judul tabel

5. tag ol digunakan untuk membuat daftar dengan penanda simbol. hampir sama seperti tag ul tapi dengan penanda simbol yang berbeda. tag ini mempunyai attribute dan value
